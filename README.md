# RetailPOS – Modern Point of Sale System

A modern, responsive Point of Sale (POS) system for retail businesses, built with **Next.js**, **React**, and **Supabase**. Easily manage sales, inventory, analytics, and more – all with a fast, clean UI and real-time updates. Works on desktop, tablet, and mobile devices.

## 🚦 Quick Start

### 1. Clone the Repository


### 2. Install Dependencies
npm install

### 3. Start the Development Server
npm run dev


### 4. Open the App

Go to [http://localhost:3000](http://localhost:3000) in your browser.

## ✨ Features

- **🛍️ POS Interface:** Manage sales, cart, and checkout seamlessly.
- **📦 Inventory Management:** Real-time product and stock tracking.
- **📊 Analytics Dashboard:** Powerful sales and business insights.
- **📱 Barcode Scanner:** Scan products quickly via your device.
- **🔐 Secure Auth:** User login via Supabase Auth.
- **🎯 Demo Mode:** Explore the system freely, no account required.
- **🌐 Offline Support:** POS works even when offline, using local storage.
- **📱 Responsive:** Beautifully adapts to desktop, tablet, and mobile.
- **🎨 Modern Theme:** Elegant dark UI with Tailwind CSS.

## 🛠️ Tech Stack

| Layer      | Stack Details                                   |
|------------|-------------------------------------------------|
| Frontend   | React 18, Next.js 14                            |
| Backend    | Supabase (PostgreSQL, Auth, Real-time DB)       |
| UI/Style   | Tailwind CSS                                    |
| State      | React Hooks                                     |
| Deployment | Vercel/Netlify ready                            |
